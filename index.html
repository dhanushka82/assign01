    Project Idea

    Overview

    The proposed program creates a weight management program intergrating a doctor personal trainer, dieticians ,and user. The user get the opportunity to live chat with dietician, and personal trainer for the updates.body sketch will be used throughout the training process. Weekly updated reports will be created till the end of the program.the user is also given chance to figure it out the suitable diet meals, exercise patterns as their life style. The ultimate goal of this program is to help the user to maintain a healthy and happy lifestyle.

    Motivation

    Gaining the figure is always easy if you are ready for a dedication. It is true that it cannot be achieved over one night, but a formal weight management program might give you a hand in achieving your super awesome figure.

    Description

    The proposed program focus on a weight management program which connects doctor, personal trainer, dieticians and user. The client enter process either through personal trainer, dietician, or doctor right after the downloading app. Those who are interested in continuing the program needs to visit doctor for medical clearance and the personal trainer or the dietician will review the user's health condition and accordingly the suitable exercises, and meals will be added the program. First month membership will be offered free of charge under the given conditions. The client gets the chance to join with personal trainer or dietician through live sessions. This might be personal or group session.

    At the begging stage client needs to feed all the relevant information such as weight, height and all body measurements. Also client needs to answer given questions about work, lifestyle, mental condition. system will create dummy sketch of the body figure and will be given other information such as body fat, body mass index. After observing dummy sketch generated by system client need to re design the figure according to their expectations. As a example some people wish to skinny body while some people wish to have muscle body. Then the program predict the time which will be taken to achieve the dream figure also program will provide another sketch under the conditions of current life style for that time duration I strongly believe by comparing different sketches it will be easy to set their mind for the program.

    Once client start the program keep track exercise, sleeping pattern, food plans. System will extract information via tracking devise which is able to detect distance travelled, heart rate, amount of fat burn. In addition producing weekly picture of user's body as a motivation factor in the journey of achieving the dream figure.

    Skills Required

      software language - be able to handle software languages very well such as java scrip and objective-c
      Cyber security -one of the most issues we having now a days cyber hacking therefore we need to make sure our app is safe to use for clients.
      Scale drawing- as we going to generate sketch of client’s figure we must know about scale drawings
      User interface design- for but up good connection between clients and app we should design app attractive and user friendly
      Back end computing- this is regarding data base management, hardware
      Businesses management- no matter how good our product is if we are unable to promote to the customers we won't be able to win over huge competition in the app market.

    Outcome

    The success of this proposed program will be view through two angles. If the user continue the maintain weight management program successfully, he/she might end up with an attractive figure and disease free life style. On the other hand, if the program runs effectively the rate of the downloading the app will accelerate the speed of the income. With development of program, marketing strategies need to be implemented and the issues in coordinating the doctor, personal trainer and dietician need to be addressed by improving the coherence.
